## Getting Started

### Prerequisites for starting up the system
* Linux:
  1. ```sh
     $ sudo apt install nodejs npm
     ```
  2. navigate to "Flight-System-Application"
  3. ```sh
     $ ./start_project.sh
     ```
* MacOs:
  1. ```sh
     $ brew install node
     ```
  2. navigate to "Flight-System-Application"
  3. ```sh
     $ ./start_project.sh
     ```
* Windows:
  1. download Node.js installer and install npm from browser
  2. navigate to "Flight-System-Application"
  3. ```sh
     $ ./start_project.bat
     ```

### Starting server
1. navigate to "Flight-System-Application"
2. ```shell
     $ ./gradlew clean build
     $ ./gradlew bootRun;
     ```

### Starting client
  1. navigate to "Flight-System-Application/src/client"
  2. ```sh
     $ npm install --force --silent
     $ npm start --silent; 
     ```
